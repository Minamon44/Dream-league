## Description
A relational database system designed to manage football leagues,
teams, players, matches, and statistics with role-based access control.

## Features
- Normalized relational schema (3NF)
- Team, player, and match management
- Admin dashboard with full CRUD operations

## Tech Stack
- Database: PostgreSQL
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
